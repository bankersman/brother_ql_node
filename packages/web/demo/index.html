Brother QL — @brother-ql/web demo

Sample UI for @brother-ql/web. WebUSB is the recommended path (Chrome, USB cable, secure context). Raw TCP via Chrome Direct Sockets (TCPSocket) is experimental — often unavailable or broken on a normal page (including many github.io tabs); you can still try it, but failures are expected. See Direct Sockets (IWA).
